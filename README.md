# Vite Starter Kit
## Features
- [x] Vite
- [x] React
- [x] TypeScript
- [x] ESLint
- [x] Prettier
- [x] Tailwind CSS
- [x] daisyUI
- [x] React Router
- [x] Firebase

## Getting Started
### Prerequisites
- [Node.js](https://nodejs.org/en/) (>= 14.0.0)

### Installation

```bash
git clone
cd vite-starter-kit
npm install
```

### Usage
#### Development
```bash
npm run dev
```

#### Production
```bash
npm run build
```

#### Lint
```bash
npm run lint
```
