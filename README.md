# RhymePage
## Description
This is a simple web application that allows users to utilize an API to suggest rhymes as they write lyrics. The application uses the [Datamuse API](https://www.datamuse.com/api/) to retrieve rhymes.

## Technologies Used
* Vite
* Firebase
* React
* TypeScript
* Tailwind CSS
* React Router
* DaisyUI
* Axios
* React Icons
